\begin{abstractzh}

长上下文推理使键值缓存（KV Cache）的显存占用随序列长度线性增长，
并逐渐成为大语言模型部署中的主要资源瓶颈。
低比特 KV Cache 量化能够直接降低缓存存储开销，
但单纯压低数值重建误差并不能保证注意力行为保持稳定：
Key 误差可能经 softmax 改写注意力分布，
Value 误差则会通过加权聚合影响输出。
因此，KV Cache 量化需要回答的不只是“能否压缩”，
还包括如何用同一组织原则连接校准、低比特恢复与预算分配。

本文提出以注意力行为为中心的行为引导统一框架。
在校准层，本文将注意力分布偏移和输出扰动作为分析对象：
对称路径使用 attention-distribution KL 建立可审计的校准代理，
角色感知低比特路径则将 Key 侧的注意力 KL 与 Value 侧的独立输出扰动代理分开处理。
这些离线校准产物进一步产生行为敏感度画像，
为后续逐层预算分配和 AutoK 扩展提供同源读数。
这种设计并不把所有路径压成同一个 KL 目标，
而是在统一工作流、产物接口和稳健选择纪律下保留 K/V 角色差异。

实验围绕三个研究问题展开。
首先，INT8 基准路径验证了行为引导校准可以形成稳定、可复现且可审计的基础保真路径；
官方 LongBench 小范围对照只作为协议一致性检查，
不被解释为覆盖广泛真实应用分布的充分证明。
其次，直接对称 INT4 会在部分模型上触发明显失稳，
且 LLaMA-3.1-8B 构成重要架构例外；
INT4-RoleAlign 在与 KIVI-style 共享非对称格式的条件下，
给出了一条从对称失效走向可用恢复的低比特实例。
最后，跨模型预算分配结果表明，
本文更稳定的经验结论不是某一策略普适最优，
而是一张由模型家族、规模与任务共同塑造的跨模型策略适用区间图谱：
Mistral-7B 是 AutoK 的最清晰正向案例，
Qwen2.5-3B 暴露早层保护回救区间，
LLaMA-3.1-8B 呈现中等规模共识区间，
Qwen2.5-14B 则表现为高性能簇共存而无稳定最优。
heuristic 的强基线地位并不削弱本文主线，
而说明某些区间的可行层集本来就很窄。

\end{abstractzh}
