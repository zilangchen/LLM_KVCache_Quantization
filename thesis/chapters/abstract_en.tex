\begin{abstracten}

This thesis starts from a failure mode that appears only after KV-cache
compression is evaluated through the attention computation itself. A
smaller cache is straightforward to obtain by lowering the bit-width,
but a small reconstruction error does not determine whether the next
softmax distribution, retrieval behavior, or task output remains stable.
The choice in this thesis is to treat attention behavior as the object
that should be preserved and audited, and to use it to connect three
decisions that are often handled separately: calibration, low-bit
recovery, and layer-wise budget allocation.

The proposed pipeline uses different behavioral proxies for different
roles in the cache. For the INT8 canonical path, the calibration
target is an attention-distribution KL proxy, which gives a compact
and reproducible fidelity instance. For the low-bit asymmetric path,
\mbox{INT4-RoleAlign} keeps the K/V distinction explicit: Key perturbations
are selected with an attention proxy, while Value perturbations
are controlled through an output proxy. The same calibration
artifacts also define layer sensitivity profiles, so the allocator and
AutoK proposer read from the same evidence rather than from a separate
hand-picked rule.

The controlled task-core and extend-profile results support a bounded interpretation.
On Qwen2.5-1.5B, the INT8 canonical path differs from FP16 by about
$+0.02$ in the three-task mean, making it the canonical fidelity case
in this thesis. In the cross-model INT4 budget band, the useful pattern
is not one cross-model optimum. Mistral-7B is the clearest AutoK case
($14.76$ core mean and $15.69$ on the extend tasks). On Qwen2.5-3B,
behavior-guided $k=1$ reaches $6.90$ on the core mean, compared with
$3.48$ for the middle-layer heuristic. On Qwen2.5-14B, uniform INT4
reaches $7.23$ core mean while AutoK-cov90 reaches $7.15$, so the top
policies remain close rather than ordered by one stable optimum. These
findings support family-/scale-/task-dependent regimes in KV-cache
quantization and allocation. Heuristic policies remain strong baselines
in some regimes, so the thesis does not claim a single allocation rule
across model families.

\keywordsen{large language models, KV-cache quantization,
behavior-guided calibration, asymmetric quantization, budget allocation}

\end{abstracten}
