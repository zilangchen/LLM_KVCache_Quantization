\begin{abstracten}

Long-context inference makes KV-cache memory grow linearly with
sequence length and has become a central deployment bottleneck for
large language models. Low-bit KV-cache quantization directly reduces
cache storage, but minimizing numerical reconstruction error alone
does not guarantee stable attention behavior: Key errors can alter the
softmax distribution, while Value errors propagate through weighted
aggregation. KV-cache quantization therefore requires more than a
lower-bit representation; it needs an organizing principle that links
calibration, low-bit recovery, and layer-wise budget allocation.

This thesis proposes a behavior-guided framework organized around
attention behavior. At the calibration layer, the framework treats
attention-distribution drift and output perturbation as the relevant
objects of analysis. Symmetric paths use attention-distribution KL as
an auditable calibration proxy, whereas the role-aware low-bit path
separates the Key-side attention-KL proxy from an independent
Value-side output-perturbation proxy. The resulting offline artifacts
also induce behavior sensitivity profiles, which are reused by the
allocator and by the AutoK extension. The framework is unified by its
workflow, artifact interface, and robust selection discipline, not by
forcing every path into a single KL objective.

The experiments answer three research questions. First, the INT8
baseline path shows that behavior-guided calibration can form a
stable, reproducible, and auditable fidelity path; the small official
LongBench check is used only as a protocol-consistency sanity check,
not as evidence of broad real-world generalization. Second, symmetric
INT4 exposes a model-dependent low-bit cliff, with LLaMA-3.1-8B as an
important architectural exception; INT4-RoleAlign then provides a
role-aware low-bit instance under the same asymmetric format family as
KIVI-style. Third, cross-model allocation within a same-order INT4
budget band is best read as a regime map rather than a universal
winner story: Mistral-7B is the clearest positive case for AutoK,
Qwen2.5-3B reveals an early-layer rescue regime, LLaMA-3.1-8B shows a
medium-scale consensus regime, and Qwen2.5-14B exhibits coexisting
high-performing clusters without a stable best policy. The strong
heuristic baseline is therefore not a failure of the framework; it
indicates that some regimes have intrinsically narrow feasible layer
sets.

\end{abstracten}
