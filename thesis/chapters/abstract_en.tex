\begin{abstracten}

As large language model context windows scale from thousands
to hundreds of thousands of tokens, KV cache memory grows
linearly with sequence length and has become the primary
bottleneck for long-context inference deployment. INT4
quantization can in principle reduce KV cache memory to
$1/4$ of its original size, yet existing methods calibrated
with numerical reconstruction error degrade sharply at long
contexts: Needle retrieval drops from 100\% to 0\% and
perplexity degrades by over 15\%. This paper revisits the
calibration objective from an attention-behavior perspective,
using the Kullback--Leibler divergence between pre- and
post-quantization attention distributions (attention-KL) as
a unified calibration and diagnostic principle, with GQA
head count $H_{kv}$ as a structural correlate linking
calibration sensitivity, degradation patterns, and fused
decoding efficiency.

This paper systematically validates the behavior-aligned
quantization framework on Qwen2.5-1.5B/7B/14B and
LLaMA-3.1-8B, yielding three core contributions.
\emph{First}, the attention-KL diagnostic lens reveals the
Key-dominated mechanism behind INT4 degradation: in
controlled diagnosis, preserving Key in FP16 recovers 93\%
of the PPL degradation on the 14B model. This finding
naturally points to a per-channel Key $+$ per-token Value
asymmetric format---the same format independently proposed
by KIVI~\cite{liu2024kivi} from engineering
observations---and supplies a mechanistic explanation for
asymmetric KV quantization.
\emph{Second}, instantiating this format through offline
attention-KL search yields \textbf{INT4-RoleAlign}, which
preserves 100\% Needle retrieval across all four models at
roughly 73\% KV cache compression, with PPL degradation
decreasing markedly with scale (1.5B: 13.7\%; 7B: 6.1\%;
8B: 2.4\%); external validity on 14B further confirms full
Needle pass rates over 4K--32K and RULER scores of
96.6\%--98.5\% over 4K--16K. The method establishes an
interpretable and reproducible offline calibration interface
for the per-channel K $+$ per-token V asymmetric format.
\emph{Third}, the Triton fused quantized decoding kernel
achieves roughly 40\% decode latency reduction at 32K in the
GQA-heavy regime ($H_{kv}=8$), which together with
INT4-RoleAlign's asymmetric compression forms an end-to-end
behavior-aligned quantization framework spanning calibration,
diagnosis, and deployment.

\end{abstracten}
